# Добро пожаловать

Это первая запись в моём блоге. Здесь я буду писать о:

- **Разработке** — технические статьи, туториалы, заметки
- **DevOps** — автоматизация, CI/CD, инфраструктура
- **AI/ML** — эксперименты с AI-инструментами и моделями
- **Проектах** — что строю и почему

## Почему именно такой сайт?

Хотелось совместить несколько вещей:

1. Терминальную эстетику, но без "хакерской" каши
2. Человечный и удобный интерфейс
3. Возможность общения — комментарии и реакции
4. Автоматический changelog из GitHub

## Технологии

Сайт построен на:

- **Astro** — SSR, отличная работа с контентом
- **React** — для интерактивных компонентов
- **Tailwind CSS** — стилизация
- **PostgreSQL** — данные
- **Drizzle ORM** — работа с БД
